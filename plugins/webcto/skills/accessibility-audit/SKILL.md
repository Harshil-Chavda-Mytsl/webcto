# Accessibility Audit

Audit for real usability, not only automated scores.

## Check

- keyboard-only navigation and visible focus
- semantic structure, landmarks, headings, buttons, links
- labels, form errors, instructions, disabled states
- contrast, motion, responsive text, target size
- ARIA correctness and screen-reader announcements

## Output

Return:

1. accessibility readiness score
2. critical blockers
3. likely WCAG impact
4. fixes with affected components
5. manual verification steps

Prefer simple semantic HTML over complex ARIA.
