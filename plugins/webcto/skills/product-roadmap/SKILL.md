# Product Roadmap

Turn ambition into sequenced delivery.

## Workflow

1. Define target user and primary workflow.
2. Split must-have, should-have, later, and avoid-for-now.
3. Sequence MVP, beta, production, growth, and enterprise phases.
4. Attach evidence needed for each major bet.
5. Add acceptance criteria and measurable outcomes.

## Output

Use:

- phase
- user outcome
- features
- risks
- validation
- exit criteria

Keep enterprise features after validated MVP unless the user gives a hard requirement.
